# 💻 Portfólio - João Eduardo (Tai)

Este é o meu portfólio profissional como desenvolvedor, criado com o objetivo de apresentar meus projetos, habilidades e evolução na área de tecnologia.

## 🚀 Sobre o projeto

Desenvolvi este site com foco em simplicidade, performance e experiência do usuário. A proposta é ter uma interface limpa e direta, onde qualquer pessoa possa navegar facilmente e conhecer meu trabalho.

O projeto foi estruturado em múltiplas páginas (Home, Sobre, Projetos e Contato), garantindo uma navegação mais organizada e profissional.

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 🎯 Objetivo

Meu objetivo com este portfólio é demonstrar na prática minhas habilidades como desenvolvedor, além de compartilhar projetos que resolvem problemas reais e contribuem para minha evolução constante.

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, reforcei conceitos importantes como:

* Estruturação de layout
* Responsividade
* Organização de código
* Boas práticas de UI/UX
